Strings

In Python, a string is a sequence of characters enclosed in quotes (single or double). 


For example:
string1 = "Hello World!"
string2 = 'Hello World!'
#Both string1 and string2 are valid strings in Python.
You can use the plus (+) operator to concatenate two strings together, like this: 

string3 = string1 + " " + string2
print(string3)
# Output: "Hello World! Hello World!"
You can also use the * operator to repeat a string a certain number of times: 

string4 = "Hello " * 3
print(string4)
# Output: "Hello Hello Hello "
Python also provides a number of built-in methods for working with strings, including:
  • len(): Returns the number of characters in a string
  • str.upper(): Converts all characters in a string to uppercase
  • str.lower(): Converts all characters in a string to lowercase
  • str.replace(old, new): Replaces all occurrences of a specified string with a new string
  • str.split(separator): Splits a string into a list of substrings based on a specified separator
  • str.join(iterable): Joins a list of strings with the string on which this method is called as a separator.
  • str.strip(): Removes leading and trailing whitespace from a string

    You can also use slicing to access a range of characters in a string, for example string1[1:4] will give ell

    You can also use string formatting to insert values into a string using placeholders, like this: 
    
name = "John"
age = 30
print("My name is {} and I am {} years old.".format(name, age))
# Output: "My name is John and I am 30 years old."
    In python 3.6 and above you can use f-strings for string formatting as well: 
    
name = "John"
age = 30
print(f"My name is {name} and I am {age} years old.")
# Output: "My name is John and I am 30 years old."
