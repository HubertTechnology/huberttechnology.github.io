Variable

In programming, a variable is a container that holds a value. Variables are used to store data that can be manipulated or used later in a program. In Python, variables are created by assigning a value to a name using the assignment operator (=). 


For example:
x = 5
#This creates a variable named x and assigns it the value 5.
The value of a variable can be changed by reassigning it a new value using the assignment operator. 

For example:
x = 5
x = 10
#Now the value of x is 10.
In Python, variables do not have a fixed type, and the type of a variable can change during the execution of the program. 

For example:
x = "Hello"
x = 5
#Here, the variable x is first assigned a string value "Hello" and then it is assigned an integer value 5.
You can also use the del keyword to delete a variable. Once a variable is deleted, it can no longer be used in the program. 

For example:
x = 5
del x


You can also use multiple assignment. 
x, y, z = 1, 2, 3
This will assign the value 1 to x, 2 to y, and 3 to z. Python has some conventions and rules for naming variables:
  • Variable names can contain letters, digits, and underscores
  • Variable names cannot start with a digit
  • Variable names cannot be the same as Python keywords (e.g., if, else, for, etc.)
  • Variable names should be descriptive and meaningful
  • Variable names should be in snake_case ( all letters in lowercase and separated by an underscore if there are multiple words)
